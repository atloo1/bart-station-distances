# bart-station-distances
distances (miles) between [BART stations](https://www.bart.gov/system-map)

### data formats
- `.json` w/ key = `'{start_station} - {end_station}'`, e.g. `'antioch - ashby'`
- `.tsv` for manual matrix lookup
